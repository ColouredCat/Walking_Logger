## Walking Logger

A handy pocket-sized device to use while hiking to track you position on a map, heading, and current weather predictions without need for a mobile phone - very handy for cheating on your DofE expeditions!

This project is part of [Hack Club Stasis](https://stasis.hackclub.com/).

### Components

- Rasperry Pi Pico microcontroller with micropython
- Ublox NEO-6M GPS
- BME280 Tempertaure, pressure and humidity sensor
- 2x SSD1306 OLED Display
- Ansman 16340 Li-ion battery

A complete bill of materials is included in PCB/material.csv.

### PCB and Schematic

The most up to date version of the PCB and schematics is included in the KiCad project in the PCB folder, as well as custom footprints and symbols for the components used freely. Pre-build gerber files are also included in the output folder.
